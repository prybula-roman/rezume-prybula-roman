Наприклад : A simple device which control level of the fuel and send data to the cloud by Mqtt protocol. Technologies: SSL/TLS, Mqtt, Json, FreeRTOS, Uart, LTE modem ...

Данную систему можно сделать, например, на базе микросхемы ESP8266. Для этого потребуется установить доп. библиотеки ‹ ESP8266WiFi.h › и библиотеку Mqtt протокола ‹ PubSubClient.h › для работы с Mqtt-брокером.

В простейшем случае нам понадобиться:

  • плата ESP8266 - 2шт.(1-издатель, 2-подписчик)

  • датчик уровня топлива - 1шт.

  • Зарегестрироваться на Mqtt-брокере
